HAUWEE'S Lava Run: Interactive Log

Inspired by classics like Jetpack Joyride, Lava Run follows our mascot Hauwee as he escapes a research center mishap using a lava lamp jetpack.

Play on Telegram

Architecture & Workflow

I architected a custom Unity WebGL pipeline focused on high-performance mobile-web compatibility. By integrating AI-assisted code generation with hand-written C# physics controllers, I achieved a fluid 60FPS experience within a 3-week development sprint.

The Creative Challenge

Bridging high-fidelity AI ideation with low-resolution 2D pixel art required a hybrid approach. I used custom AI prompts for mascot concepts, then manually hand-refined every frame in Aseprite to ensure a consistent, authentic retro-gaming aesthetic.

Technical Objective

The core goal was to create an interactive engagement layer for our project token ($UBP). I implemented a custom reward system that bridges in-game performance with external tokenized incentives, creating a true Web3-enabled gaming loop.
